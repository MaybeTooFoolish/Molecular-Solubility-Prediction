# Molecular Solubility Prediction

## Research Question

Can molecular structure descriptors predict aqueous solubility, and which molecular features matter most chemically?

## Goal

This project is my introduction to computational chemistry trying to apply the advanced math and chemistry that I have self-learned. I am learning Python, molecular representations, RDKit descriptors, data analysis, graphing, and basic machine learning through a solubility prediction project.

## Current Status

- Started with SMILES strings as molecular representations
- Learning how to represent molecules in Python
- Next step: calculate molecular descriptors using RDKit

## Tools

- Python
- pandas
- matplotlib
- RDKit
- scikit-learn

## Project Roadmap

1. Represent molecules with SMILES strings
2. Calculate molecular descriptors
3. Load an experimental solubility dataset
4. Graph descriptor-solubility relationships
5. Train simple prediction models
6. Analyze model errors chemically
7. Write a final report
